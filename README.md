# Geospatial Data Analysis

## Project Overview
This project analyzes location-based sales data to identify high-demand regions and potential areas for business expansion.

## Objectives
- Visualize sales geographically
- Detect high-demand areas
- Identify low business presence regions
- Support expansion decision-making

## Features
- Data cleaning and preprocessing
- Interactive maps
- Heatmap visualization
- Regional sales analysis
- Business expansion insights

## Technologies Used
- Python
- Pandas
- Matplotlib
- Folium
- Geopandas
